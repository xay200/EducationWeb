== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugin > Add new'.
2. Search for `thim elementor kit`.
3. Activate LearnPress from your Plugins page.

**From WordPress.org**
1. Search, select and download `thim-elementor-kit`.
2. Upload the `thim-elementor-kit` folder to the `/wp-content/plugins/` directory or alternatively upload the thim-elementor-kit.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
